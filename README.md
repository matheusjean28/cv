## Run Project

Before copy this Project open the folter and run the follow command

#### Install dependencies 
```http
  npm install 
```

 --- 
 
#### Run Develop 
```http
  npm run dev 
```
Supose that you have fineshi your changes, make sure to do an inspection by whole code, remove unnecessaries files. 

 --- 

#### Run Build 
```http
  npm run build
```
